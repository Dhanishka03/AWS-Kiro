# SmartKitchen AI – Quick Meal Planner

SmartKitchen AI is a simple and user-friendly web application that helps users reduce daily cooking stress by suggesting recipes based on the ingredients they already have at home. It also includes a weekly meal planner and favorites feature to organize meals efficiently.

---

## Features

* Ingredient-Based Recipe Suggestions
  Enter available ingredients and get matching recipes instantly.

* Recipe Cards
  View recipe name, ingredients, steps, and images in a clean UI.

* Weekly Meal Planner
  Plan your meals for the entire week (Monday–Sunday).

* Favorites System
  Save your favorite recipes using localStorage.

* Reduce Food Waste
  Highlights how many ingredients match to encourage efficient usage.

---

* HTML – Structure

## How It Works

1. User enters available ingredients
2. App matches them with recipes from the JSON file
3. Recipes are ranked based on ingredient match
4. Users can:

   * View recipes
   * Add to favorites
   * Assign meals to weekly planner

---

## How to Run

1. Download or clone this repository
2. Open `index.html` in your browser
3. Start adding ingredients and explore recipes

No server or backend setup required.
